Detailed Description

Provides a method for strategies to calculating the maximum range of a Starship.

The range of a ship is specified in Megalights (MGLT) and is calculated as the total number of hours that the ship can provide consumables for the entire crew, multiplied by the maximum number of MGLTs it can fly per hour. Where the nescessary information to calculate this is not present, implementations should return a null value.
